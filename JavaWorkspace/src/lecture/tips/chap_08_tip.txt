### Chap_08 추상클래스와 인터페이스 tip ###

# 추상 클래스
- 아직 완성되지 않은 class
- 추상 메소드가 있을 수도 있고, 없을 수도 있음
- 추상 메소드가 있던 없던간에 추상 클래스 객체는 생성할 수 없음
- 추상 메소드가 있으면 상속받는 자식 클래스에서 반드시 구현해야함
- 추상 메소드를 구현한 자식 클래스가 있으면 자식 클래스로 객체를 생성할 수 있음

# 인터페이스
- 다중상속 가능
- interface 에서 정의한 모든 변수는 public static final 로 정의된다
- interface 에서 정의한 모든 메소드는 public abstract 로 정의된다
