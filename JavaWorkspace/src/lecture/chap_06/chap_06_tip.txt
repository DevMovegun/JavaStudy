### Chap_06 메소드 tip ###

# 메소드 이름
- 메소드 이름은 변수명과 이름짓기가 비슷함. 처음엔 소문자
- 보통 메소드는 동작을 수행하므로 동사를 많이 사용

# 메소드 오버로딩
- 전달되는 매개변수의 타입이 다르거나 갯수가 다를 때 사용
- 메소드 이름을 같게 만들어도 됨
- 반환형은 모두 같아야 함
- 메소드 내에서 오버로딩된 다른 메소드를 호출 할 수 있음.

# 메소드가 필요한 이유
- 코드의 불필요한 중복을 줄일 수 있다

# 메인 메소드로 매개변수 값 넘기기
- 오른쪽 상단에 Current File -> Edit Configurations 또는 메뉴바에 Run -> Edit Configurations
- Add Configurations -> Application -> 전달하려는 java class 선택 -> 밑에 줄에 전달하고자 하는 매개변수 값 입력
- 각 매개변수는 " " 으로 구분
